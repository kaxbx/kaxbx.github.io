Pacman

  • first let to do update to system
     sudo pacman -Syyu
-S, --sync
-y, --refresh
-U, --upgrade
  • let install some fresh packages
     sudo pacman -S sudo man-db i3-gaps mpv mpd ncmpcpp mpc python3 ruby golang rofi chromium firefox dmenu
man-db -> pages of info about application
i3-gaps -> window manager
mpv -> video player from local and online
mpd -> local music server
ncmpcpp -> gui in terminal control local music server
mpc -> terminal control music server
python3 -> language very easy to lean
ruby -> language easy to lean
golang -> language easy to learn
rofi -> it like launcher
chromium -> browser without google stuff
firefox -> browser
dmenu -> launcher in top

zip

  • -r recursive directory
  • -x exclude
    zip -r app.zip app -x "*/.git/*" "*.DS_Store*" "*/node_modules/*"

tar

  • -c Create a new archive containing the specified items.
  • -t List archive contents to stdout.
  • -x Extract to disk from the archive.
  • -j (c mode only) Compress the resulting archive with bzip2
  • -z (c mode only) Compress the resulting archive with gzip
  • -Z (c mode only) Compress the resulting archive with compress
tar -zcvf app.tar.gz app
tar -jcvf app.tar.bz2 app
tar -Jcvf app.tar.xz app

ffmpeg

# aiff to mp3
ffmpeg -i hello.aiff -f mp3 -acodec libmp3lame -ab 19200 -ar 44100 hello.mp3
# 2 hours blank video
ffmpeg -t 7200 -f lavfi -i color=c=black:s=640x480 -c:v libx264 -tune stillimage -pix_fmt yuv420p output.mp4
# cut video
ffmpeg -i movie.mp4 -ss 00:00:03 -t 00:00:08 -async 1 cut.mp4
# alternative
ffmpeg -ss 00:01:12 -i movie.mp4 -ss 00:00:00 -t 00:00:03 -c copy cut.mp4
# merge video and audio
ffmpeg -i video.mp4 -i audio.wav -c:v copy -c:a aac output.mp4
# extract audio from mp4
ffmpeg.exe -i in.mp4 -vn -c:a copy -ac 2 out.m4a
# extract audio from webm
ffmpeg -i "source.webm" -vn -acodec copy "output.opus"
